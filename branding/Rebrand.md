Transform the app into:

- soft

---

## 🎨 DESIGN SYSTEM

Use this EXACT design system:

Colors:

- Primary: #0F3D3E (petrol)
- Secondary: #8ECDB8 (sage)
- Accent: #FF8A7A (coral, use sparingly)
- Background: #F6F4EF (warm)
- Card: #FFFFFF (slightly warm white)
- Text primary: #1C1C1E
- Text secondary: #6B6B6E

Style:

- soft gradients (very subtle)
- rounded corners (16–20pt)
- no harsh borders
- minimal shadows
- lots of whitespace

DO NOT:

- use red error colors
- use medical icons
- use sharp edges
- use dense layouts

Use SF Pro (system font)

Hierarchy:

- Title: large, airy

- Body: very readable
- Avoid dense text blocks
